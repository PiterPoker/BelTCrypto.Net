# BelTCrypto.Net

**BelTCrypto.Net** — это высокопроизводительная реализация национального стандарта Республики Беларусь **СТБ 34.101.31-2020 (belt)** на платформе .NET. 

Проект включает в себя полную реализацию всех алгоритмов шифрования, хеширования и управления ключами, определенных в стандарте.

## Стек технологий
* **Платформа:** .NET 8.0+ / .NET 9.0
* **Язык:** C# 12/13
* **Оптимизация:** `Span<T>`, `ReadOnlySpan<T>`, `stackalloc` (Memory-safe, Low-allocation).
* **Тестирование:** NUnit 4 (верификация по эталонным векторам Приложения А).

## Реализованные алгоритмы
Библиотека реализует **все** компоненты стандарта:
* **Базовые:** `belt-block` (128/192/256 бит), `belt-hash`, `belt-compress`.
* **Режимы:** `ECB`, `CBC`, `CFB`, `CTR`, `MAC`.
* **Комбинированные (AEAD):** `belt-dwp`, `belt-che`.
* **Управление ключами:** `belt-keyexpand`, `belt-keyrep`, `belt-kwp`.
* **Специальные:** `belt-bde`, `belt-sde` (дисковое шифрование), `belt-fmt` (форматное).

## Как запустить

### 1. Требования
Убедитесь, что у вас установлен [.NET SDK 8.0](https://dotnet.microsoft.com/download) или выше.

### 2. Клонирование и сборка
```bash
git clone https://github.com/PiterPoker/BelTCrypto.Net.git
cd BelTCrypto.Net
dotnet build
```

### 3. Запуск тестов
Верификация алгоритмов по официальным таблицам СТБ:
```bash
dotnet test
```
